# Widget Type Catalog

122 widget types available for use in widgetized pages and PDPs. All widgets belong to the `storefront` group.

> **Tip:** Use `list_widget_types(search: "keyword")` to search by name, and `get_widget_schema(widget_type: "TYPE_NAME")` to get the full JSON Schema with all fields and validation rules.

## Widget Structure

Every widget in a page config follows this structure:

```json
{
  "id": "unique-widget-id",
  "type": "BANNER",
  "widgetData": { },
  "header": { },
  "layout": { }
}
```

### How to use this catalog

1. **Find the right widget type** — scan this catalog or use `list_widget_types(search: "keyword")`
2. **Get the full schema** — call `get_widget_schema(widget_type: "TYPE_NAME")` for exact field requirements
3. **Construct the widget object** — follow the schema for `widgetData` fields
